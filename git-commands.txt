git init
  - initializing (starting) a git project
  - allows git version control

git status
  - shows us which files have been changed 
  - where the files are at in the process of adding them to the history

git add 
  - moves our changes to the staging area

git commit
  - saves the changes to the history

git checkout .
  - undo changes that have not been added to the staging area

git reset
  - moves changes from staging area to local changes

git revert
  - changes code to the previous commit

git log
  - displays commit history
